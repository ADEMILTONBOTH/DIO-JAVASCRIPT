O que são Tags.

As tags são elementos essenciais na linguagem HTML. Vamos explorar o que elas são e como funcionam:

Definição de Tags:

Uma tag (ou etiqueta) é usada para marcar ou identificar algo em um documento HTML.

Assim como uma etiqueta de preços no mercado, as tags no HTML marcam elementos. Elas determinam onde um elemento começa e onde ele termina. Sintaxe das Tags: As tags são sempre delimitadas pelos sinais de chevron (< e >). Existem dois tipos de tags: Tags que precisam de fechamento: Possuem a sintaxe . Tags que não precisam de fechamento: Têm a estrutura .

Atributos:

Os atributos personalizam as tags, modificando sua estrutura ou funcionalidade.

Alguns atributos comuns incluem:

  • class: Atribui uma classe ao elemento.
  • id: Atribui um ID único ao elemento.
  • style: Permite incluir estilos CSS dentro da tag.
  • lang: Define o idioma principal do elemento.
  • title: Define o título do elemento.
  • alt: Define um texto alternativo para imagens (importante para SEO).
  • hidden: Oculta o elemento.
  • align: Define o alinhamento (exemplo: right, center, left).
  • width e height: Definem largura e altura para o elemento.
